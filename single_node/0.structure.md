
### Single Node
```
k3s/
├── deployments/
│   ├── webapp-deployment.yaml
│   ├── api-deployment.yaml
│   └── db-deployment.yaml
├── services/
│   ├── webapp-service.yaml
│   ├── api-service.yaml
│   └── db-service.yaml
├── ingress/
│   └── ingress-rules.yaml
├── config/
│   ├── configmap.yaml
│   └── secret.yaml
└── autoscaling/
    └── webapp-hpa.yaml
```

### Multi Node
```
k3s/
├── apps/
│   ├── webapp/
│   │   ├── deployment.yaml
│   │   ├── service.yaml
│   │   ├── ingress.yaml
│   │   ├── configmap.yaml
│   │   └── hpa.yaml
│   ├── api/
│   │   ├── deployment.yaml
│   │   ├── service.yaml
│   │   ├── hpa.yaml
│   │   └── secret.yaml
│   └── db/
│       ├── deployment.yaml
│       ├── service.yaml
│       └── configmap.yaml
└── global/
    ├── ingress-controller.yaml
    ├── storageclass.yaml
    └── namespace.yaml
```
